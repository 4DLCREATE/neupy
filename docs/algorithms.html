Layers

Input and hidden layers
Class name Description Has derivative?
LinearLayer Layer with linear activation function. No
SigmoidLayer Layer with sigmoid activation function. Yes
StepLayer Layer with step activation function. No
TanhLayer Layer with tanh activation function. Yes
RectifierLayer Layer with rectifier activation function. No
SoftplusLayer Layer with softplus activation function. Yes
SoftmaxLayer Layer with softmax activation function. Yes
EuclideDistanceLayer Layer output equal to Euclide distance between input value and weights. No
AngleDistanceLayer Layer which output equal to cosine distance between input value and weights. No
Output layers
Class name Description
