Home

NeuPy is a Python library for Artificial Neural Networks.
You can run and test different Neural Network algorithms.

Dependencies
------------

* Python 2.7, 3.4
* Theano >= 0.8.1
* NumPy >= 1.9.0
* SciPy >= 0.14.0
* Matplotlib >= 1.4.0
