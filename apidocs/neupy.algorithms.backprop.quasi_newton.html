neupy.algorithms.backprop.quasi_newton module

class neupy.algorithms.backprop.quasi_newton.QuasiNewton(connection, options=None, **kwargs)[source]

Bases: neupy.algorithms.backprop.backpropagation.Backpropagation

Quasi-Newton Backpropagation algorithm optimization.

Parameters:

update_function : {‘bfgs’, ‘dfp’, ‘psb’, ‘sr1’}

Update function. Defaults to bfgs.

h0_scale : float

Factor that scale indentity matrix H0 on the first iteration step. Defaults to 1.

gradient_tol : float

In the gradient less than this value algorithm will stop training procedure. Defaults to 1e-5.

optimizations : list or None

The list of optimization algortihms. None by default. If this option is not empty it will generate new class which will inherit all from this list. Support two types of optimization algorithms: weight update and step update.

use_raw_predict_at_error : bool

If the value is specified as a True, it means that learning algorithm will use the results obtained by the network without passing through the output layer. Defaults to False.

step : float

Learns step, defaults to 0.1.

show_epoch : int

This option controls how offen you’d see the epoch updates information in the terminal, defaults to 1

shuffle_data : bool

If it’s True class shuffles all your training data before training your network, defaults to True.

error : function

Function which controls your training error. defaults to mse

use_bias : bool

Uses bias in the network, defualts to True.

train_epoch_end_signal : function

Calls this function when train epoch finishes.

train_end_signal : function

Calls this function when train process finishes.

verbose : bool

Verbose output for network. Defaults to True. False disables output.

Backpropagation

Examples

Simple example

>>> import numpy as np
>>> from neupy import algorithms
>>>
>>> x_train = np.array([[1, 2], [3, 4]])
>>> y_train = np.array([[1], [0]])
>>>
>>> qnnet = algorithms.QuasiNewton(
...     (2, 3, 1),
...     update_function='bfgs',
...     verbose=False
... )
>>> qnnet.train(x_train, y_train)
