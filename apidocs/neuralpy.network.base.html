neuralpy.network.base module

class neuralpy.network.base.BaseNetwork(connection, **options)[source]

Bases: neuralpy.core.base.BaseSkeleton, neuralpy.network.base.NetworkSignals

Base class Network algorithms.

Parameters:

step : float

Learns step, defaults to 0.1.

show_epoch : int

This option controls how offen you’d see the epoch updates information in the terminal, defaults to 1

shuffle_data : bool

If it’s True class shuffles all your training data before training your network, defaults to True.

error : function

Function which controls your training error. defaults to mse

use_bias : bool

Uses bias in the network, defualts to True.

train_epoch_end_signal : function

Calls this function when train epoch finishes.

train_end_signal : function

Calls this function when train process finishes.

verbose : bool

Verbose output for network. Defaults to True. False disables output.

Methods

plot_errors(use_semilog=False) Draws the error rate update plot. It always shows network learning progress. When you add cross validation data set into training function it displays validation data set error as separated curve. If parameter use_semilog is equal to the True value it displays x-axis in logarithmic scale.
last_error() Returns the last error network result after training procedure or None value if you try to get it before network training.
