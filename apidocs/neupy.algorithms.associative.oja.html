neupy.algorithms.associative.oja module

class neupy.algorithms.associative.oja.Oja(**options)[source]

Bases: neupy.network.learning.UnsupervisedLearning, neupy.network.base.BaseNetwork

Oja unsupervised algorithm which minimize feature space.

Parameters:

minimized_data_size : int

Expected number of features after minimization, defaults to 1

weights : array-like or None

Predefine default weights which controll your data in two sides. If weights are, None before train algorithms generate random weights. Defaults to None.

step : float

Learns step, defaults to 0.1.

show_epoch : int

This option controls how offen you’d see the epoch updates information in the terminal, defaults to 1

verbose : bool

Verbose output for network. Defaults to True. False disables output.

train_epoch_end_signal : function

Calls this function when train epoch finishes.

train_end_signal : function

Calls this function when train process finishes.
Raises:

ValueError

  • Try reconstruct without training.
  • Invalid number of input data features for train and reconstruct methods.

Notes

  • In practice use step as very small value. For example 1e-7.

Examples

>>> import numpy as np
>>> from neupy import algorithms
>>>
>>> data = np.array([[2, 2], [1, 1], [4, 4], [5, 5]])
>>>
>>> ojanet = algorithms.Oja(
...     minimized_data_size=1,
...     step=0.01,
...     verbose=False
... )
>>>
>>> minimized = ojanet.train(data, epsilon=1e-5)
>>> minimized
array([[-2.82843122],
       [-1.41421561],
       [-5.65686243],
       [-7.07107804]])
>>> ojanet.reconstruct(minimized)
array([[ 2.00000046,  2.00000046],
       [ 1.00000023,  1.00000023],
       [ 4.00000093,  4.00000093],
       [ 5.00000116,  5.00000116]])
