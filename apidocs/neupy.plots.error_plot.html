neupy.plots.error_plot module

neupy.plots.error_plot.error_plot(network, logx=False, ax=None, show=True)[source]

Makes line plot that shows training progress. x-axis is an epoch number and y-axis is an error.

Parameters:

logx : bool

Parameter set up logarithmic scale to x-axis. Defaults to False.

ax : object or None

Matplotlib axis object. None values means that axis equal to the current one (the same as ax = plt.gca()). Defaults to None.

show : bool

If parameter is equal to True plot will instantly shows the plot. Defaults to True.
Returns:

object

Matplotlib axis instance.

Examples

>>> from neupy import algorithms, plots
>>>
>>> gdnet = algorithms.GradientDescent((2, 3, 1))
>>> gdnet.train(x_train, y_train, x_test, y_test, epochs=100)
>>>
>>> plots.error_plot(gdnet)
